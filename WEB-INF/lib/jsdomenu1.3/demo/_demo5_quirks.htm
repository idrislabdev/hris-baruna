Unlike absolute menus, the position of static menus "flows" with the document structure. For example, suppose you place a static menu after a paragraph, when you resize the browser, the height of the paragraph changes, but the static menu will still be after the paragraph.

A static menu (using <div id="staticMenu"></div>) has been placed after the following paragraph. Resizing the browser does not change the relative position of the menu. It will still be positioned after the paragraph.
