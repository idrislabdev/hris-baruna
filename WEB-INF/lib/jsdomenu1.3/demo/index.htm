Demo « jsDOMenu

Click on any of the following links to view the demo:

  • Demo 1 (strict, quirks): Cursor menus, absolute menus and fixed menus all in one page.
  • Demo 2 (strict, quirks): Change the theme of the menus without reloading the page.
  • Demo 3 (strict, quirks): Horizontal menu bar, with the help from jsDOMenuBar.
  • Demo 4 (strict, quirks): Use filters to control where the pop-up menu can show up.
  • Demo 5 (strict, quirks): Static menus, whose position on the page "flows" with the document structure.
  • Demo 6 (strict, quirks): Show an icon before the display text of the menu item.

You can also view the demo in frames.
