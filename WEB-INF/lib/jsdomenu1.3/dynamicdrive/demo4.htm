Dynamic Drive Static positioned jsDomenu demo

This example illustrates jsDomenu set up to:

  • Be static positioned on the page. Static means "inline" on the page, so it appears precisly where you put it on the page, and flows with the rest of the content.
  • Use an Win XP theme (simply by pointing to a different CSS file).
  • Note: When setting the menu to static, the "id" parameter of::
    menuObj = new jsDOMenu(width, mode, id, alwaysVisible, className);
    must be set. See /docs/reference.html for more info.

Check out the configuration file ("ddconfig4.js") to see how this is set up, and view the source of this page to see the necessary codes to insert on your page.

Obviously, be sure to also check out the other folders outside of "dynamicdrive" for comprehensive documentation and demos as provided by the author himself.

Enjoy!
