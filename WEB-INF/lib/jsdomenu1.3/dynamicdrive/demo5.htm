Dynamic Drive Absolutely positioned icon jsDomenu demo

This example illustrates jsDomenu set up to:

  • Be absolutely positioned on the page- x and y coordinates.
  • Use an Win XP theme (simply by pointing to a different CSS file).
  • Set up an alternate, user selectable style sheet in Mozilla/NS7+. If you're using one of those browsers, simply click on the icon at the bottom left of your browser to instantly change the look of this menu.
  • Displays an icon image alongside certain menu items.

Check out the configuration file ("ddconfig5.js") to see how this is set up, and view the source of this page to see the necessary codes to insert on your page.

Obviously, be sure to also check out the other folders outside of "dynamicdrive" for comprehensive documentation and demos as provided by the author himself.

Enjoy!
