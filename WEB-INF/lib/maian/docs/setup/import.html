» HTML SEARCH IMPORT TOOL:
The search import tool is a useful little function to enable to you batch add pages. It only supports .html or .htm file extension pages at this time.

Please follow the instructions below to use the import tool.
» HOW TO USE THE IMPORT TOOL:
1 => Firstly, open up the 'import_tool.php' file in a text editor and update the database connection information at the top of the file to point to the database that holds your search tables. This will be the same as the information in your 'inc/db_connection.inc.php' file.

2 => Next, upload the 'import_tool.php' file into any directory on your server that has .html or .htm pages you want to import into the search system.

3 => Once you have done this, access the file in your browser wherever it is located.

4 => Follow the import instructions on screen to import your pages. You can specify which pages you want to import using the checkboxes you will see next to each page. You can also specify additional keywords and descriptions on import.

5 => The import file will work in any directory on your server.

Enjoy!
